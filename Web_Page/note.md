### Web page structure

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <header>헤더에 해당되는 콘텐츠를 여기에 둔다.</header>

    <nav>내비게이션 메뉴는 여기에 둔다.</nav>

    <aside id="left">상품 카테고리 메뉴는 여기에 둔다.</aside>

    <section>
      <article>상품에 대한 정보를 여기서 보여준다.</article>
    </section>

    <aside id="right">쇼핑카드와 로그인 정보를 보여준다.</aside>

    <footer>각주에 해당되는 콘텐츠는 여기에 둔다.</footer>
  </body>
</html>
```
